
## Description

A NestJs TypeScript website for gitHub authorization.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
## Usage
Once the server is running, you can access the application in two ways:

Localhost: Open your browser and navigate to [localhost:3000](http://localhost:3000/) to view the application.
Vercel: The application is also deployed on Vercel and can be accessed at https://true-foundry-two.vercel.app/.
